Start Game

Once everyone has joined and submitted some names, click below to start the game.

There will be 2 teams. Each player's turn will last 60 seconds.

Round 1

In Round 1, you can use as many words as you need to describe the name you draw.

In Round 2, you can only use a single word. If it helps, you can say it multiple times.

In Round 3, you cannot make a sound! You must act out the name instead.
